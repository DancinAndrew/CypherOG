# Popping & Funk Styles 風格分類體系

> 個人整理，系統性描述 Popping 文化中的所有主要流派、其技術特徵、歷史淵源和代表人物。

## Popping — 主流派詳解

### 1. Boogaloo
- **起源**：1970s 加州 Richmond / Fresno，Sam Solomon（Boogaloo Sam）創立
- **技術特徵**：
  - 全身性律動，以臀部和膝蓋為中心的圓形搖擺
  - 動作流暢（fluid）而非爆裂，比其他 Popping 子流派更「液態」
  - 身體的每個關節都參與律動，製造蠕動感（rolling movement）
  - 腿部常見 dip（身體下沉）和 roll（旋轉）
- **文化地位**：Popping 的源頭技術，所有 Popper 必學
- **代表人物**：Boogaloo Sam、Skeeter Rabbit（Michael Wayne Davis）
- **容易混淆**：Boogaloo 是 Popping 的子流派，不是 Popping 的同義詞；Boogaloo 更強調流動性，Popping 更強調爆點清晰度

### 2. Waving（波浪）
- **起源**：Electric Boogaloos 早期技術，後被全球 Popper 廣泛採用
- **技術特徵**：
  - 動作像波浪或電流從身體一端傳到另一端
  - Arm Wave（手臂波）：從指尖開始，波浪經過手腕、肘、肩傳遍全身
  - Body Wave（身體波）：全身垂直或水平方向的波動
  - 需要精確的 Isolation 控制
- **代表人物**：Salah（以 Waving 著稱，特別是全身波和手指波）
- **難度等級**：中高，全身 Waving 需要大量練習

### 3. Tutting（圖坦卡門）
- **起源**：靈感來自古埃及 Tutankhamun 法老壁畫中的側面姿勢，美國街舞文化中發展
- **技術特徵**：
  - 手臂和手指形成直角（90°）或方形幾何圖案
  - 動作以直線和方形為美學基礎，與 Waving 的曲線形成對比
  - 細分：Finger Tutting（手指）、Arm Tutting（手臂）、Body Tutting（全身）
  - 每個動作轉換都需要精確定格（Hit）
- **代表人物**：Nonstop（Marquese Scott）擅長結合 Tutting 和 Animation

### 4. Animation（動畫）
- **起源**：受卡通和定格動畫啟發，在 Popping 文化中獨立發展
- **技術特徵**：
  - 模仿卡通角色或定格動畫的移動方式
  - 核心技術：Strobing（閃爍）、Slow Motion（慢動作）、Dime Stop（硬停）的組合
  - 讓觀眾感覺舞者在「逐格播放」
  - 需要極強的身體控制力
- **代表人物**：Marquese Scott（NONSTOP）是最知名的 Animation 舞者
- **網路影響**：NONSTOP 的 YouTube 影片是讓 Animation 在全球普及的關鍵

### 5. Turfing
- **起源**：2000s 初期，加州奧克蘭（Oakland）地下社群
- **技術特徵**：
  - 融合 Popping、Gliding（滑步）、Mime（默劇）元素
  - 強調講故事（storytelling）和地方文化認同
  - 常有戲劇性的情感表達
  - 地板動作豐富
- **縮寫含義**：TURFERS = Taking Up Room For Everyone Right
- **文化背景**：Oakland 街頭文化的產物，代表底層社區的自我表達
- **代表人物**：Mistah F.A.B.、Keak da Sneak（說唱歌手兼推廣者）

### 6. Robot（機器人）
- **起源**：Popping 文化早期發展，受科幻電影和機器人形象影響
- **技術特徵**：
  - 完全模仿機器人的移動方式
  - 所有動作都是直線的，避免有機曲線
  - 精準的角度控制（90°轉向）
  - Dime Stop 之後的姿勢保持
- **代表人物**：Nate "Robot" Johnson（Electric Boogaloos 成員，以 Robot 風格命名）

### 7. Liquid / Liquid Popping
- **技術特徵**：
  - 融合 Waving 的流動性和 Popping 的爆點
  - 動作在「液態」和「爆發」之間不斷切換
  - 是現代 Popping 表演中最常見的融合風格
  - 需要同時掌握流動性控制和爆點技術

### 8. Strutting
- **技術特�征**：
  - 強調步伐的節奏感和風格化
  - 以走路動作為基礎，融入 Funk 音樂節奏
  - 在 1970s 加州社群中很普遍

### 9. Electric Boogaloo（風格名）
- **注意**：Electric Boogaloo 既是一個**舞團名稱**（Electric Boogaloos），也是一個**風格名稱**
- **作為風格**：結合 Boogaloo 的流動性和 Popping 的爆點，是整個 Popping 文化的代名詞
- **作為團體**：Electric Boogaloos = 1977 年由 Boogaloo Sam 在 Fresno 成立的舞團

## Locking（兄弟流派）

### 核心技術
- **Lock**：突然靜止在一個誇張姿勢，然後快速釋放繼續律動
- **特徵動作**：Wrist Roll（腕部旋轉）、Points（指向動作）、Scooby Doo（轉身）、Leo Walk（特色步伐）
- **音樂關係**：比 Popping 更緊密依附 Funk 音樂的 downbeat
- **與 Popping 的差異**：Locking 節奏感更強、動作更外向（outgoing），Popping 則更注重技術控制

## 評分標準詳解（Battle 裁判視角）

根據 HHI 官方規則和 Juste Debout 評分標準，Popping Battle 的評判維度：

| 評分維度 | 說明 | 重要性 |
|---|---|---|
| **Musicality（樂感）** | 動作與音樂的節奏、旋律、歌詞的對應程度 | ★★★★★ 最重要 |
| **Foundation（基礎）** | Popping 基本動作的扎實程度（Pop 的清晰度、Isolation 的精準度） | ★★★★★ |
| **Originality（原創性）** | 動作語彙是否有個人特色，而非複製他人 | ★★★★ |
| **Execution（執行）** | 動作的完成度，技術是否乾淨清晰 | ★★★★ |
| **Technique（技術）** | 特定技術動作（如 Waving、Tutting）的掌握程度 | ★★★★ |
| **Creativity（創造力）** | 動作組合和即興創作的多樣性 | ★★★ |
| **Floor Work（地板動作）** | 是否有效運用地板，但在 Popping 中比 Breaking 少見 | ★★ |
| **Showmanship（表演力）** | 與觀眾的互動、眼神、舞台魅力 | ★★★ |
| **Costume（服裝）** | 服裝是否與風格相符（在部分賽事中計入） | ★ |

## 常見混淆澄清

| 問題 | 答案 |
|------|------|
| Popping 和 Boogaloo 是同一種舞嗎？ | 不完全是。Boogaloo 是 Popping 的子流派，更強調流動性；Popping 是包含 Boogaloo 在內的更大類別。 |
| Popping 和 Locking 是同一種舞嗎？ | 不是，但兩者常被合稱為 Funk Styles。Locking 由 Don Campbell 在 LA 獨立創立，Popping 由 Boogaloo Sam 在 Fresno 創立。 |
| Breaking（霹靂舞）是 Popping 嗎？ | 不是。Breaking 是 Hip-Hop 四大元素之一（DJ、MC、Graffiti、Breaking），與 Popping/Locking 是完全不同的流派，雖然常在同一文化圈共存。 |
| 「Electric Boogaloo」是風格還是團體？ | 兩者都是。Electric Boogaloos 是 1977 年成立的舞團；Electric Boogaloo 也指他們發展的舞蹈風格。 |
| Tutting 和 Popping 有什麼關係？ | Tutting 是 Popping 的子流派，通常與 Popping 技術結合使用（帶有爆點的幾何動作）。 |
