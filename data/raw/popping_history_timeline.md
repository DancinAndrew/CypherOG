# Popping & Funk Style 街舞歷史時間線

> 個人整理，涵蓋 1960s 至今的重要事件。資料來源：多份 Wikipedia 條目、舞者訪談、社群記錄。

---

## 1960s — 文化背景奠基

| 年份 | 事件 |
|------|------|
| 1964 | James Brown 發行 "Out of Sight"，奠定 Funk 音樂的節奏基礎。 |
| 1965 | Sly & The Family Stone 在舊金山成立，將 Funk 與 Rock、Soul 融合，影響後來的舞蹈文化。 |
| 1968 | James Brown 發行 "Say It Loud – I'm Black and I'm Proud"，Funk 音樂成為非裔美國文化認同的象徵。 |
| 1960s末 | Don Campbell 在洛杉磯開始發展 Locking 動作，早期稱為 Campbellock。 |

---

## 1970s — Popping 的誕生（黃金年代）

| 年份 | 事件 |
|------|------|
| 1970 | Parliament-Funkadelic（George Clinton 主導）成立，成為 Funk 音樂最具影響力的樂團之一。 |
| ~1970 | Don Campbell 在加州聖塔莫尼卡城市大學（LACC）開始教學 Locking，逐漸形成風格。 |
| 1972 | The Lockers 舞團成立（Don Campbell、Toni Basil、Shabba-Doo 等），在電視節目 Soul Train 首次亮相。 |
| ~1974 | **Sam Solomon（Boogaloo Sam）**在加州 Fresno 開始發展 Boogaloo 和 Popping 技術，融合身體律動與爆點。 |
| ~1975 | Timothy Solomon（Popin' Pete）和其他舞者加入，Electric Boogaloos 前身形成。 |
| ~1977 | **Electric Boogaloos 正式成立**（Fresno, California），成員包括 Boogaloo Sam、Popin' Pete、Skeeter Rabbit、Sugar Pop、Dane 1、Nate Robot 等。成為 Popping 文化最重要的源頭組織。 |
| 1978 | Electric Boogaloos 在洛杉磯開始演出，透過電視節目（如 Soul Train）將 Popping 帶給全國觀眾。 |
| 1979 | Sugarhill Gang 發行 "Rapper's Delight"，標誌 Hip-Hop 音樂進入主流。Hip-Hop 文化開始與 Funk Styles 並行發展。 |

---

## 1980s — Popping 進入主流媒體

| 年份 | 事件 |
|------|------|
| 1980 | Electric Boogaloos 與 Michael Jackson 開始合作，Boogaloo Sam 和 Popin' Pete 參與 MJ 演唱會和音樂錄影帶的編舞。 |
| 1983 | **Michael Jackson 在 Motown 25 週年紀念演出中表演 Billie Jean**，其中包含 Moonwalk（受 Electric Boogaloos 影響）。這場表演讓 Street Dance 文化大爆光。 |
| 1984 | 電影 **"Breakin'"** 上映，Electric Boogaloos 成員（Boogaloo Sam、Popin' Pete）出演，讓 Popping 和 Breaking 進入好萊塢主流。 |
| 1984 | 電影 **"Beat Street"** 上映，進一步推廣 Hip-Hop 和 Street Dance 文化。 |
| 1984 | 電影 **"Breakin' 2: Electric Boogaloo"** 上映，片名直接致敬 Electric Boogaloos。 |
| 1985 | Funk Styles 在媒體曝光達到高峰，但隨後 Hip-Hop 和 New Jack Swing 逐漸取代其在主流音樂中的地位。 |
| 1980s末 | Breaking、Locking、Popping 三大 Street Dance 文化開始在歐洲（特別是法國）傳播，形成獨立的歐洲 Funk 社群。 |

---

## 1990s — 地下社群深化，歐洲崛起

| 年份 | 事件 |
|------|------|
| ~1990 | 法國 Funk Styles 社群開始成形，巴黎成為歐洲 Popping 的重鎮。Salah、Lilou 等法國舞者日後將引領國際賽事。 |
| 1993 | **Battle of the Year** 創立（德國），早期 Breaking 為主的國際賽事，帶動歐洲 Street Dance 文化交流。 |
| ~1995 | **Hip Hop International（HHI）** 前身活動開始，後發展為全球最大 Street Dance 錦標賽。 |
| 1998 | **Juste Debout 前身活動**在巴黎舉辦（由 Bams 和 Anne Nguyen 組織），早期聚焦 Popping 和 Locking。 |

---

## 2000s — 全球賽事體系建立

| 年份 | 事件 |
|------|------|
| 2002 | **Juste Debout 正式成為年度賽事**（巴黎），成為全球最重要的 Popping/Locking/House 2v2 比賽，吸引來自 60+ 個國家的舞者。 |
| 2003 | **Hip Hop International（HHI）** 正式成立，總部設於美國，舉辦年度世界錦標賽（World Hip Hop Dance Championship）。 |
| ~2003 | **Turfing** 在加州奧克蘭地下社群興起，結合 Popping、Gliding 和地方文化認同。 |
| 2005 | 電視節目 **"America's Best Dance Crew（ABDC）"** 開播（後於 2008 正式播出），帶動 Street Dance 在美國年輕世代的普及。 |
| 2008 | **STEEZY Studio** 的前身概念開始發展（正式成立約 2015 年），線上舞蹈教學平台崛起。 |
| ~2009 | **Marquese Scott（NONSTOP）** 開始在 YouTube 發布 Animation 和 Slow Motion 影片，在全球網路上引發廣泛迴響，單支影片觀看次數逾千萬。 |

---

## 2010s — 數位化與全球化

| 年份 | 事件 |
|------|------|
| 2010 | YouTube 成為舞蹈文化傳播的主要平台，Popping 知識開始大規模數位化。 |
| ~2012 | **Salah（法國）** 贏得多個 Juste Debout 冠軍，被廣泛認為是最具影響力的當代 Popper 之一。 |
| 2015 | **STEEZY Studio** 正式成立（洛杉磯），提供高品質線上舞蹈課程，包含 Popping、House、Breaking 等多種風格。 |
| ~2016 | **Funk in Focus Online** 成立，專為 Popping 和 Funk Styles 舞者提供線上教育資源。 |
| 2018 | **Red Bull Dance Your Style** 系列賽事在全球多個城市開啟，以「觀眾投票」取代傳統裁判制度，引發 Street Dance 社群對評分方式的討論。 |
| ~2018 | **Urban Artistry** 啟發了多個街舞教育組織，包括 Funk in Focus Online，強調文化傳承和舞蹈教育。 |
| 2019 | **IMD（Inspire Movement Dance）** 因在打擊飢餓方面的貢獻獲得美國國會認可，代表舞蹈社群的社會影響力持續增長。 |

---

## 2020s — 後疫情時代與 Olympics

| 年份 | 事件 |
|------|------|
| 2020 | COVID-19 疫情迫使 Juste Debout、HHI 等大型賽事暫停，線上 Cypher 和數位社群迅速成長。 |
| 2021 | **Breaking（霹靂舞）** 正式加入 2024 巴黎奧運項目，帶動所有 Street Dance 文化的國際能見度。 |
| 2024 | **Paris Olympics 2024** 首次將 Breaking 納入奧運競技，雖然 Popping 未入選，但此事件提升了整體 Street Dance 的全球關注度。 |
| 2025 | Juste Debout、HHI、Red Bull Dance Your Style 等賽事持續舉辦，Popping 社群在亞洲（台灣、日本、韓國）持續壯大。 |

## 重要地理據點

| 地區 | 特色 |
|------|------|
| **Fresno, CA** | Popping / Boogaloo 誕生地，Electric Boogaloos 的搖籃 |
| **Richmond / Oakland, CA** | Boogaloo 早期發展地，Turfing 發源地 |
| **Los Angeles, CA** | Locking 誕生地，Hollywood 媒體曝光中心 |
| **Paris, France** | 歐洲 Funk Styles 重鎮，Juste Debout 主辦地 |
| **Tokyo, Japan** | 亞洲最大 Popping 社群之一，舉辦多項國際賽事 |
| **Taipei, Taiwan** | 台灣 Popping 社群中心，定期舉辦 Battle 活動（如 Taiwan Bboy City 的 Popping 組別） |
